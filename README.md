# Research paper - GPU Architecture & High-Performance Computing
 
-------------------------------------
Description
-------------------------------------
This repository contains my research paper focused on **GPU Architecture, Parallel processing models, and their applications in high-performance computing**. The paper explores how GPUs accelerate computation, compares CPU vs. GPU design philosophies, and discusses modern advancements in GPU technology

-------------------------------------
Contents
-------------------------------------
- Full research paper
- Reference & Citations
- Supporting notes and materials

-------------------------------------
Research focus
-------------------------------------
- Evolution of GPU Architecture
- Parallel computing & SIMD/SIMT models
- GPU vs. CPU performance characteristics
- Application in AI, simulation, and scientific computing

-------------------------------------
Purpose
-------------------------------------
This paper was completed as part of academic work in Computer Science, demonstrating an understanding of modern GPU design and its impact on high-performance computing and real-world applications.
